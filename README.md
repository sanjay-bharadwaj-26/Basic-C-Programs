# Basic C Programs 🖥️

A collection of beginner-level C programs covering fundamental concepts.

## Programs Included

| File | Description |
|------|-------------|
| basics.c | Variables, data types, I/O |
| Print Hello.c | Hello World |
| Simple Calculator.c | +, -, *, / operations |
| Palindrome.c | Check if number/string is palindrome |
| Armstrong.c | Armstrong number check |
| Prime or not.c | Prime number checker |
| Leap year.c | Leap year logic |
| Swapping & character.c | Variable swapping |
| Days to Seconds.c | Unit conversion |
| Pairs & Divisors.c | Divisors of a number |
| Small & Large.c | Min/Max finder |
| Sum of N natural numbers.c | Loops and series |

## How to Run
```bash
gcc filename.c -o output
./output
```
